# FINANCIAL-GUARDIAN
🛡️ Financial Guardian — AI Powered FinTech Security App

Financial Guardian is a modern AI-powered financial management and fraud detection web app designed to help users track finances, detect fraud risks, manage debts, and improve financial health — all in one secure dashboard.

Built with a sleek UI, multilingual support, and AI assistance, this project demonstrates a complete FinTech ecosystem prototype.
